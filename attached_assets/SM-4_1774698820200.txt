# INSTRUCTION POUR REPLIT AI

## Module 01.4 — Brand Guidelines

### Objectif

Modifier l'application Neo Branding Studio pour implémenter un système de génération de chartes graphiques **ultra-précises** pour le sous-module Brand Guidelines, en utilisant l'exemple parfait fourni comme **golden reference**.

L'objectif est de garantir que **chaque génération** respecte exactement la structure PDF, les 10 règles graphiques (R01–R10) avec leurs 4 blocs (Description, Do's, Don'ts, Conséquences), les spécifications de mise en page A4, et les contraintes d'accessibilité WCAG, quels que soient les paramètres d'entrée.

---

## 1. Contexte

**Fichier actuel à modifier** :  
`api-server/src/routes/openai/enhance-prompts.ts` (section Module 01)

**Données d'entrée du formulaire** :
```typescript
interface BrandBrief {
  brandName: string;           // ex: "Theravectys"
  sector: string;              // ex: "tech", "luxury", "streetwear", "fitness", "cosmetic"
  tone: string;                // ex: "professionnel", "audacieux", "minimaliste"
  values: string[];            // ex: ["innovation", "rigueur", "fiabilité", "expertise", "engagement"]
  logo: {
    primary: string;           // description du logo principal
    variations: string[];      // descriptions des variantes
  };
  palette: {
    primary: string;           // ex: "#0B1F3B"
    secondary: string;         // ex: "#12B5B0"
    accent: string;            // ex: "#B7F000"
    neutrals: {
      dark: string;            // ex: "#101820"
      medium: string;          // ex: "#546A7B"
      light: string;           // ex: "#E9EEF2"
      white: string;           // ex: "#FFFFFF"
    };
  };
  typography: {
    heading: string;           // ex: "Space Grotesk"
    body: string;              // ex: "Inter"
    code: string;              // ex: "JetBrains Mono"
  };
}
```

**Sortie attendue** : Un prompt structuré qui générera une charte graphique complète au format PDF avec les 10 règles graphiques.

### 2.3. Fichier : `api-server/src/prompts/modules/module-01-4-guidelines/prompt-builder.ts`

Ce fichier construit le prompt final en remplaçant les parties variables de l'exemple parfait.

```typescript
import { goldenExample } from './golden-example.json';
import { sectorAdaptations, wcagPairs } from './mapping-rules.json';

interface BrandBrief {
  brandName: string;
  sector: string;
  tone: string;
  values: string[];
  logo: {
    primary: string;
    variations: string[];
  };
  palette: {
    primary: string;
    secondary: string;
    accent: string;
    neutrals: {
      dark: string;
      medium: string;
      light: string;
      white: string;
    };
  };
  typography: {
    heading: string;
    body: string;
    code: string;
  };
}

interface RuleBlock {
  description: string;
  dos: string[];
  donts: string[];
  consequences: string;
}

export function buildGuidelinesPrompt(brief: BrandBrief): string {
  // Récupérer les adaptations selon le secteur
  const adaptations = sectorAdaptations[brief.sector] || sectorAdaptations.tech;
  
  // Construire la description des valeurs
  const valuesText = brief.values.join(', ');
  
  // Construire le prompt final
  let prompt = goldenExample.goldenExample;
  
  // Remplacer les variables de base
  prompt = prompt.replace(/Theravectys/g, brief.brandName);
  prompt = prompt.replace(/marque \*\*tech\*\*/g, `marque **${brief.sector}**`);
  prompt = prompt.replace(/ton \*\*professionnel\*\*/g, `ton **${brief.tone}**`);
  prompt = prompt.replace(/innovation, rigueur scientifique, fiabilité, expertise, engagement/g, valuesText);
  
  // Remplacer les spécifications de la charte
  prompt = prompt.replace(/\*\*Logo\*\* : proposer un \*\*logo principal\*\* \+ \*\*3 variantes\*\*/g, 
    `**Logo** : ${brief.logo.primary} + ${brief.logo.variations.join(', ')}`);
  
  // Remplacer les couleurs
  prompt = prompt.replace(/#0B1F3B/g, brief.palette.primary);
  prompt = prompt.replace(/#12B5B0/g, brief.palette.secondary);
  prompt = prompt.replace(/#B7F000/g, brief.palette.accent);
  prompt = prompt.replace(/#101820/g, brief.palette.neutrals.dark);
  prompt = prompt.replace(/#546A7B/g, brief.palette.neutrals.medium);
  prompt = prompt.replace(/#E9EEF2/g, brief.palette.neutrals.light);
  prompt = prompt.replace(/#FFFFFF/g, brief.palette.neutrals.white);
  
  // Remplacer les typographies
  prompt = prompt.replace(/Space Grotesk/g, brief.typography.heading);
  prompt = prompt.replace(/Inter/g, brief.typography.body);
  prompt = prompt.replace(/JetBrains Mono/g, brief.typography.code);
  
  // Générer les 10 règles avec adaptations
  const rules = generateRules(brief, adaptations);
  
  // Remplacer chaque règle
  for (let i = 1; i <= 10; i++) {
    const ruleKey = `R0${i}` as keyof typeof rules;
    const ruleContent = rules[ruleKey];
    const rulePattern = new RegExp(`### R0${i} — .*?(?=### R0${i+1}|---)`, 's');
    prompt = prompt.replace(rulePattern, ruleContent);
  }
  
  // Générer le tableau WCAG
  const wcagTable = generateWcagTable(brief);
  prompt = prompt.replace(/### R06 — Accessibilité WCAG 2\.1 AA.*?(?=### R07)/s, wcagTable);
  
  // Ajouter la page Quick specs
  const quickSpecs = generateQuickSpecs(brief);
  prompt = prompt.replace(/Quick specs.*?(?=\n\n\[PARAMÈTRES)/s, quickSpecs);
  
  // Ajouter les instructions de mise en page PDF
  prompt += `

## INSTRUCTIONS DE MISE EN PAGE PDF

**Format**: A4 portrait (210×297mm)
**Marges**: 16mm
**Grille**: 8pt
**Police titres**: ${brief.typography.heading}
**Police corps**: ${brief.typography.body}
**Police code**: ${brief.typography.code}

**Structure du document**:
- Page 1: Couverture (${brief.brandName} Brand Guidelines)
- Page 2: Sommaire
- Page 3: Vision rapide et valeurs de marque
- Pages 4-13: Règles R01 à R10 (une page par règle)
- Page 14: Quick specs et contacts

**Éléments visuels**:
- Utiliser ${brief.palette.neutrals.light} comme fond des pages
- Utiliser ${brief.palette.primary} pour les titres de section
- Utiliser ${brief.palette.secondary} pour les accents et appels visuels
- Iconographie linéaire, stroke 2px, angles arrondis 2-3px
- Images: abstractions data/science, éclairage contrôlé

**Export**:
- PDF print-ready, 300dpi
- Polices embarquées
- Pas de watermark
- Compression JPEG minimale
`;
  
  return prompt;
}
```
