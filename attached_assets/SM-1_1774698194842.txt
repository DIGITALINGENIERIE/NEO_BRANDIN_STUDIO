# INSTRUCTION POUR REPLIT AI

## Module 01.1 — Logo Generator

### Objectif

Modifier l'application Neo Branding Studio pour implémenter un système de génération de prompts **ultra-précis** pour le sous-module Logo Generator, en utilisant l'exemple parfait fourni comme **golden reference**.

L'objectif est de garantir que **chaque génération** respecte exactement la structure, le niveau de détail et les contraintes techniques de l'exemple, quels que soient les paramètres d'entrée (nom de marque, secteur, valeurs, etc.).

### 2.3. Fichier : `api-server/src/prompts/modules/module-01-1-logo/prompt-builder.ts`

Ce fichier construit le prompt final en remplaçant les parties variables de l'exemple parfait.

```typescript
import { goldenExample } from './golden-example.json';
import { sectorMappings, toneMappings } from './mapping-rules.json';

interface BrandBrief {
  brandName: string;
  sector: string;
  tone: string;
  values: string[];
  logoStyle?: string;
}

export function buildLogoPrompt(brief: BrandBrief): string {
  // Récupérer les mappings selon le secteur
  const sectorMapping = sectorMappings[brief.sector] || sectorMappings.tech;
  
  // Adapter le style selon le secteur
  let style = sectorMapping.style;
  let ambiance = sectorMapping.ambiance;
  let symbolConcept = sectorMapping.symbolConcept;
  
  // Si l'utilisateur a spécifié un style, le prioriser
  if (brief.logoStyle) {
    style = brief.logoStyle;
  }
  
  // Construire la description des valeurs
  const valuesText = brief.values.join(', ');
  
  // Adapter les couleurs selon le secteur
  const primaryColor = sectorMapping.primaryColor;
  const secondaryColor = sectorMapping.secondaryColor;
  const accentColor = sectorMapping.accentColor;
  
  // Adapter le symbole si nécessaire (pour streetwear, fitness, etc.)
  let symbolDescription = `Créer un symbole abstrait à gauche du wordmark, évoquant à la fois ${symbolConcept}:`;
  
  if (brief.sector === 'tech') {
    symbolDescription += `
- Icône basée sur un "V" stylisé construit avec 2 segments géométriques (épaisseur uniforme), formant une flèche vectorielle discrète vers l'avant (innovation) et une impression de stabilité (base plus large, sommet maîtrisé).
- Intégrer une micro-grille implicite via 2–3 points nodaux (petits cercles) alignés sur une trajectoire, comme une visualisation scientifique propre.
- Style: lignes pleines, angles propres, aucune référence médicale clichée.`;
  } else if (brief.sector === 'luxury') {
    symbolDescription += `
- Icône basée sur un monogramme stylisé évoquant l'héritage et la précision.
- Lignes épurées, géométrie parfaite, symétrie maîtrisée.
- Style: lignes fines, élégance discrète, sans surcharge décorative.`;
  } else if (brief.sector === 'streetwear') {
    symbolDescription += `
- Icône basée sur une lettre stylisée ou un pictogramme urbain audacieux.
- Trait épais, angles vifs, présence marquée.
- Style: impact visuel fort, typographique, authenticité.`;
  } else if (brief.sector === 'fitness') {
    symbolDescription += `
- Icône basée sur un symbole de mouvement (flèche, vague, dynamique).
- Lignes énergiques, sensation de vitesse et de puissance contrôlée.
- Style: dynamique, motivant, performance.`;
  } else if (brief.sector === 'cosmetic') {
    symbolDescription += `
- Icône basée sur une forme organique (goutte, pétale, cercle parfait).
- Courbes douces, fluidité, sensation de pureté.
- Style: naturel, apaisant, élégance subtile.`;
  }
  
  // Construire le prompt final en remplaçant les variables
  let prompt = goldenExample.goldenExample;
  
  // Remplacer les variables
  prompt = prompt.replace(/Theravectys/g, brief.brandName);
  prompt = prompt.replace(/tech corporate/g, `${brief.sector} ${brief.tone}`);
  prompt = prompt.replace(/innovation, la rigueur scientifique et la fiabilité/g, valuesText);
  prompt = prompt.replace(/stabilité, confiance et précision/g, `${brief.values[0] || 'stabilité'}, ${brief.values[1] || 'confiance'}, ${brief.values[2] || 'précision'}`);
  prompt = prompt.replace(/Style corporate minimaliste, sobre et stable/g, `Style ${style}`);
  prompt = prompt.replace(/Ambiance "trusted technology": sérieux, moderne, non ostentatoire/g, `Ambiance "${ambiance}"`);
  prompt = prompt.replace(/évoquant à la fois \*\*vecteurs\*\*, \*\*trajectoires contrôlées\*\* et \*\*rigueur scientifique\*\*/g, `évoquant ${symbolConcept}`);
  prompt = prompt.replace(/Icône basée sur un \*\*"V" stylisé\*\* construit avec 2 segments géométriques.*?(?=\n\n)/s, symbolDescription.split('\n').slice(1).join('\n'));
  prompt = prompt.replace(/#003087/g, primaryColor);
  prompt = prompt.replace(/#6C757D/g, secondaryColor);
  prompt = prompt.replace(/#2F80ED/g, accentColor);
  prompt = prompt.replace(/bleu marine confiance/g, `${primaryColor} (couleur primaire)`);
  prompt = prompt.replace(/gris corporate neutre/g, `${secondaryColor} (couleur secondaire)`);
  prompt = prompt.replace(/tech highlight/g, `highlight ${brief.sector}`);
  
  // S'assurer que les 4 variations utilisent les bonnes couleurs
  prompt = prompt.replace(/fond #FFFFFF, icône et wordmark en #003087/g, `fond #FFFFFF, icône et wordmark en ${primaryColor}`);
  prompt = prompt.replace(/fond #0B1220, wordmark et icône en #FFFFFF/g, `fond #0B1220, wordmark et icône en #FFFFFF`);
  prompt = prompt.replace(/blanc #FFFFFF sur fond bleu marine #003087/g, `blanc #FFFFFF sur fond ${primaryColor}`);
  
  // Ajouter la liste NEGATIVE_PROMPT spécifique au secteur
  const negativePrompt = `vieux, obsolète, rétro, vintage, désordonné, watermark, 3D, textures, artefacts IA${brief.sector === 'luxury' ? ', cheap, bas de gamme, imitation' : ''}${brief.sector === 'streetwear' ? ', commercial, corporate, trop propre' : ''}${brief.sector === 'fitness' ? ', statique, mou, sans énergie' : ''}`;
  prompt = prompt.replace(/vieux, obsolète, rétro, vintage, désordonné, câbles apparents en désordre, interface confuse, erreurs visibles, bugs, désinvolte, informel, trop décontracté, argot, watermark, mockup photoréaliste, gradients agressifs, 3D, bevel\/emboss, textures, bruit, glitch, texte illisible, kerning mauvais, compression JPEG visible, artefacts IA, formes incohérentes, détails trop fins non-scalables, symboles médicaux clichés, ADN, seringue, croix, mascotte, cartoon, low-res/g, negativePrompt);
  
  return prompt;
}
```
